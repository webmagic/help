1. ログオン¶

WebMagic へ接続するための画面です。

1.1. ログオンする¶

  • ユーザーのパソコンで Web ブラウザを立ち上げ、 WebMagic の URL を指定します。 [1]

  • 表示されたログオン画面にて、「ログオン ID」,「パスワード」,「グループ ID」を入力します。 [2] [3]

Note

大文字と小文字は区別されます。

  • ログオンをクリックします。

  • 次のような画面が表示されたら、ログオン成功です。

1.2. ログオフする¶

メニューの をクリックすると、ログオフできます。

Note

ブラウザを ☓ ボタンで閉じてもログオフされません。

脚注

[1]WebMagic の URL については、御社 WebMagic ご担当者様までお問い合わせください。
[2]WebMagic のログオン ID とパスワードとグループ ID については、御社 WebMagic ご担当者様までお問い合わせください。
[3]パスワードは機密保持のため、**** マークで表示されます。
